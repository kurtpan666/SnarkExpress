Test Instructions

  3. Wait 3-5 seconds for preview to load
  4. If preview appears, the issue is with the main page

Telegram Debugging Tips

  • Use @WebpageBot in Telegram to force refresh cache
  • Try: Send URL to @WebpageBot, it will show you what Telegram sees
  • Clear preview: Delete and resend the link
  • Telegram caches for ~24 hours, be patient

Common Telegram Issues

  • Image must be accessible (no 404)
  • Image must be > 200x200px
  • Description shouldn't be too long (keep under 200 chars)
  • HTTPS required (HTTP won't work)
  • No redirects on image URL
